*   Проверка конвергенции (сведения лучей/пикселей).

### 4. Анализ насыщенности (Saturation)

Выбор оптимальных точек насыщенности для калибровки:

| Уровень насыщенности | Характеристика этапа |
| :---: | :--- |
| **40%** | Минимальный порог (нижняя граница диапазона) |
| **60%** | Среднее значение (шаг +20% от начальной точки) |
| **80%** | Максимальный рекомендуемый уровень |

> **💡 Примечание к выбору:**
> Исходя из доступных данных, наиболее корректными точками являются **40%** и **60%**.
> Значение **74%** может служить верхней границей (как ближайшее доступное к 80%). Разница между 74% и 60% составляет 14%, что немного меньше идеального шага в 20%, но все же обеспечивает достаточный визуальный контраст.
> *Данные актуальны для всех четырех цветовых каналов.*

### 5. Параметр Pixel Persistence

**Определение:**
Термин **Pixel Persistence** (инерционность или время отклика пикселя) характеризует скорость изменения состояния пикселя.

**Интерпретация значения 400 пикселей/сек:**
Данная метрика указывает на пропускную способность управления строкой: за одну секунду контроллер способен адресовать и обновить состояние 400 пикселей в последовательности. Это напрямую влияет на скорость сканирования матрицы и отсутствие шлейфов при динамичном изображении.

### 6. Анализ тестовой таблицы принтера

**Задача:** Выделить диапазон значений насыщенности, при котором оттенки визуально неотличимы от белого фона.

**Ход анализа:**
При рассмотрении градиентной шкалы (от 0% до 100%):
1.  **0%** — чистый белый цвет (фон).
2.  **10%** — крайне светлый оттенок, различимый с трудом.
3.  **20%** — пограничное значение; для большинства наблюдателей оттенок еще сливается с фоном.
4.  **30% и выше** — цвет становится явно выраженным и легко читается.

**Вывод:**
Диапазон неразличимых значений: **0% – 20%**.
*Пояснение:* В этом интервале контраст недостаточен для уверенного восприятия информации, так как оттенки сливаются с белой подложкой.

## Контрольные вопросы

### Тема: Классификация мониторов и устройство LCD-панелей

#### 1. Типы мониторов

Основным критерием разделения мониторов является технология формирования изображения.

*   **ЭЛТ (CRT - Cathode Ray Tube):** Устаревшая технология на основе электронно-лучевой трубки. Изображение формируется пучком электронов, возбуждающим люминофор.
    *   *Плюсы:* Отличная цветопередача, углы обзора.
    *   *Минусы:* Габариты, вес, высокое энергопотребление, ЭМИ.
*   **ЖК (LCD - Liquid Crystal Display):** Наиболее массовый тип. Использует слой жидких кристаллов для модуляции света от подсветки.
    *   *Плюсы:* Компактность, экономичность, безопасность.
*   **Плазменные (PDP):** Используют свечение газа в ячейках под действием тока.
    *   *Особенности:* Глубокий черный цвет, высокая контрастность. Применялись преимущественно в ТВ большого формата из-за стоимости и энергопотребления.
*   **OLED (Organic LED):** Передовая технология, где каждый пиксель является самостоятельным источником света.
    *   *Преимущества:* Идеальный черный цвет (пиксель выключен), минимальная толщина, высочайшая контрастность.
*   **Прочие виды:** Проекционные, лазерные, виртуальные ретинальные (VRD).

**Дополнительная классификация:**
*   По типу сигнала: аналоговые, цифровые.
*   По интерфейсу: VGA, DVI, HDMI, DisplayPort.
*   По размерности: 2D, стереоскопические (3D).

#### 2. Устройство ЖК-монитора

Конструкция LCD-экрана представляет собой многослойный «сэндвич». Рассмотрим слои от источника света к зрителю:

1.  **Модуль подсветки (Backlight):**
    *   Исторически использовались лампы CCFL.
    *   Современный стандарт — **LED**. Бывает двух видов:
        *   *Edge-LED:* Диоды по краям (тонкие корпуса).
        *   *Direct-LED:* Матрица диодов за экраном (возможность локального затемнения).
2.  **Нижний поляризатор:** Пропускает свет только определенной плоскости колебаний.
3.  **TFT-матрица (Стекло с транзисторами):** Содержит тонкопленочные транзисторы для управления каждым субпикселем. Создает электрическое поле.
4.  **Слой жидких кристаллов:** Молекулы кристаллов меняют ориентацию под напряжением, поворачивая плоскость поляризации света.
5.  **Цветовой фильтр (Color Filter):** Стекло с нанесенными RGB-фильтрами. Окрашивает проходящий свет. Черная маска (Black Matrix) разделяет субпиксели.
6.  **Верхний поляризатор:** Расположен перпендикулярно нижнему. Блокирует или пропускает свет в зависимости от того, насколько его повернули кристаллы.

**Принцип действия:**
Напряжение на TFT управляет поворотом кристаллов -> изменяется количество прошедшего через второй поляризатор света -> свет окрашивается фильтрами -> формируется итоговое изображение.

#### Основные типы матриц

| Тип матрицы | Особенности | Сфера применения |
| :--- | :--- | :--- |
| **TN+film** | Высокое быстродействие, низкая цена. Плохие углы обзора и цветопередача. | Бюджетные офисные и киберспортивные мониторы. |
| **IPS** | Превосходная цветопередача, широкие углы обзора (178°). Выше время отклика и цена. | Дизайн, обработка фото/видео, профессиональная работа. |
| **VA (MVA/PVA)** | Золотая середина. Высокая контрастность, глубокий черный. Углы обзора лучше TN, но хуже IPS. | Мультимедиа, домашние кинотеатры, универсальное использование. |
