# Лабораторная работа № 6
## Определение основных характеристик и тестирование видеосистемы ПК

**Тема программы:** Видеосистема персонального компьютера.  
**Цель работы:** Изучить современные видеокарты на графических процессорах NVIDIA, AMD (ATI) и технологии объединения видеокарт.  
**Время выполнения:** 1 час  
**Оборудование:** учебный персональный компьютер.  
**Программное обеспечение:** операционная система, презентация, тестовые программы.

---

## Теоретические основы

Для выполнения данной лабораторной работы выбраны две утилиты: **GPU-Z v0.6.0** и **FurMark v1.10.2**. GPU-Z предоставляет информацию о видеокарте, а FurMark проводит её стресс-тестирование.

### GPU-Z
Программа для вывода подробной информации о графическом адаптере, поддерживающая видеокарты как NVIDIA, так и AMD (ATI). С её помощью можно:
- Узнать точную модель видеокарты;
- Определить интерфейс подключения (PCI-E, AGP и др.);
- Получить данные о графическом процессоре: версия BIOS, ревизия чипа, частоты в 2D/3D-режимах и при разгоне, поддержка версий DirectX;
- Просмотреть характеристики видеопамяти: тип, объём, разрядность шины.

### FurMark
Утилита для стресс-тестирования видеокарт, создающая экстремальную нагрузку на графический процессор с помощью визуализации «волосатого бублика» (torus). Позволяет оценить стабильность системы охлаждения и максимальную производительность под нагрузкой.

---

## Порядок выполнения работы

1. Скачать и установить программу для тестирования видеокарты (например, GPU-Z или аналог).
2. С помощью **GPU-Z** определить все параметры тестируемой видеокарты и зафиксировать их в отчёте:
   - Имя графического адаптера;
   - Технология изготовления (техпроцесс);
   - Версия BIOS;
   - Поддерживаемая версия DirectX;
   - Объём видеопамяти (Memory Size);
   - Тип видеопамяти;
   - Версия драйвера;
   - Частота графического процессора (GPU Clock).
3. Провести тестирование видеокарты с помощью **FurMark v1.10.2**:
   - При запуске настроить параметры тестирования в соответствии с образцом;
   - Запустить тест с визуализацией «бублика» (глазное яблоко);
   - Дождаться окончания тестирования. Если тест не завершится за 10 минут — зафиксировать результат из левого верхнего угла и завершить тест нажатием клавиши **ESC**;
   - Записать результаты в FPS: минимальный, средний, максимальный.
4. Повторить тест в FurMark, отключив визуализацию «бублика» (нажатием пробела). Записать новые показатели FPS.
5. Установить и запустить программу **FluidMark**.
6. Настроить параметры теста и запустить его. В процессе выполнения в левом верхнем углу отображается информация о прохождении теста. Через 5 минут зафиксировать:
   - Показатели FPS;
   - Температуру видеочипа.
7. Сделать вывод о качестве и производительности установленного видеоадаптера.

## 7. Вывод о качестве установленного видеоадаптера

**Intel HD Graphics 2000** — очень слабый видеоадаптер начального уровня.

### ❌ Недостатки:
1. Крайне низкая производительность — всего 2 FPS в тесте FluidMark при разрешении 1366×768;
2. Устаревшая технология — техпроцесс 32 нм (2011 год), всего 6 исполнительных блоков (EU);
3. Отсутствие официальной поддержки драйверов для Windows 10;
4. Производительность ниже 1% от современных дискретных видеокарт;
5. Низкие базовые частоты — 650–850 МГц с возможностью boost до 1000–1100 МГц.

### ⚠️ Назначение:
- Подходит **только** для базовых задач: офисная работа, просмотр видео, веб-браузинг;
- Непригодна для современных игр и ресурсоёмких графических приложений;
- Интегрированное решение начального уровня эпохи Sandy Bridge.

### 📊 Оценка: 3/10  
*(Крайне слабый адаптер для современных задач)*

## Контрольные вопросы

### 1. Какие другие программы используют для тестирования видеокарт?

| Программа | Тип | Назначение |
| :--- | :--- | :--- |
| **3DMark** | Комплексный бенчмарк | Тесты DirectX 11/12, сравнение с другими системами, онлайн-рейтинги |
| **Unigine Heaven / Superposition** | Стресс-тест и бенчмарк | Красивая визуализация, проверка стабильности при нагрузке |
| **FurMark** | Стресс-тест | Максимальная нагрузка на GPU, проверка системы охлаждения |
| **MSI Afterburner** | Мониторинг и разгон | Отображение FPS, температур, напряжений в реальном времени |
| **GPU-Z** | Диагностика | Подробная информация о видеокарте: частоты, память, BIOS, датчики |
| **OCCT** | Комплексное тестирование | Тесты GPU, CPU, памяти; поиск ошибок и нестабильности |
| **PassMark PerformanceTest** | Общий бенчмарк системы | Включает 2D/3D-тесты видеокарты, сравнение с базой данных |
| **Catzilla / Fire Strike** | Игровые бенчмарки | Ориентированы на игровую производительность в разных разрешениях |
